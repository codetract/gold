# GCT

GCT is a standard Ethereum token fully backed by physical gold. Refer to https://gold.codetract.io for more information.

## For developers

#### Ethereum mainnet

GCT address: `0x560c5528ff9886d83Ae117845B180E6DCF6B5175`
